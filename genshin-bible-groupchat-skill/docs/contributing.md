# 贡献与扩展指南

## 新增档案模板

新增梗档案时，建议按下面格式写：

```md
## 档案二十：【档案名称】

### 梗本体

> “核心梗句 1” / “核心梗句 2”

### 触发条件 / 关键词

- 当群友提到……
- 当群聊讨论……

### 底层逻辑与情绪状态

- **情绪**：……
- **逻辑**：……

### 拓展变体

- **【轻度版】**：“……”
- **【重度版】**：“……”
- **【跨界升华版】**：“……”
```

## 写作建议

- 梗句要短，方便模型改写。
- 触发词要明确，方便检索。
- 变体不要全都写成长篇，保留短打版本。
- 尽量用“反串”而不是“真实攻击”。
- 不要加入开盒、人肉、线下威胁、现实报复等内容。
- 不要加入未公开版本爆料和内鬼内容。

## 输出风格建议

一个好用的群聊 Skill 应该像“会接梗的群友”，而不是“每次都复制圣经的小作文机器”。

推荐：

```text
差不多得了，你的声音太尖锐了。跳过剧情就是跳过人生，这点道理还要我教吗？
```

不推荐：

```text
我冷笑一声，捂住耳朵，用高高在上的语气说道：……
```
