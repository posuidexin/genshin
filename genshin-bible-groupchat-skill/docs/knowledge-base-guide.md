# 知识库搭配指南

这个 Skill 的定位是“梗库和语气库”，不是完整的原神资料库。建议单独建立一个 Markdown / Office 文档知识库，用于存放正式设定和事实内容。

## 推荐知识库内容

建议按下面的主题拆成多个 Markdown 文档：

```text
01_世界观与核心术语.md
02_国家地区与文化标签.md
03_角色组织与阵营索引.md
04_玩法系统与游戏机制.md
05_剧情任务节日与记录.md
06_中文社区语境梗与争议词典.md
07_Skill联动规则.md
08_检索问答种子与触发样例.md
09_来源与维护清单.md
```

## 分工原则

```text
事实问题 → 查知识库
玩梗/吐槽/对线 → 调用 Skill
事实 + 玩梗 → 先查知识库，再用 Skill 语气加工
```

例子：

| 用户输入 | 推荐处理 |
|---|---|
| “芙宁娜是谁？” | 查知识库，正常回答角色设定 |
| “芙宁娜凭什么 560 亿？” | 调用 Skill 的金融神童梗 |
| “海灯节是什么？” | 查知识库回答节日设定 |
| “春节没年味了” | 调用 Skill 的赛博宗祠梗 |
| “散兵剧情争议是什么？” | 查知识库，必要时轻度抽象 |
| “老玩家又在骂散兵了” | 调用老玩家阻碍/饭圈提纯相关档案 |

## 知识库写法建议

每个条目尽量使用适合检索的短卡片：

```md
# 芙宁娜

## 类型
角色 / 枫丹 / 水元素 / 剧情核心人物

## 简介
芙宁娜是《原神》枫丹主线剧情中的重要角色，与枫丹审判、神明身份、预言危机等剧情高度相关。

## 适合联动的 Skill 梗
- 金融神童：560 亿 IP 价值
- 文化输出：国产角色价值
- 尖锐声音：当群友质疑角色强度或剧情时
```

## 不建议放入知识库的内容

- 未公开、内鬼、泄露、测试服爆料。
- 大段未经整理的用户评论区内容。
- 无来源的版本爆料和争议截图。
- 过长的原文搬运。

知识库越干净，Bot 越不容易把玩梗内容当事实。
